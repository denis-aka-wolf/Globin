## Введение

В языке Globino для образования множественного числа используется простое и последовательное правило. В отличие от многих национальных языков, где существуют исключения и различные формы множественного числа, в Globino используется единое окончание для всех частей речи.

## Правило образования множественного числа

Для образования множественного числа в Globino к слову добавляется окончание **-zo** и **-zon**.

**Примеры:**
    - domo → domozo (дом → дома)
    - libro → librozo (книга → книги)
    - kato → katozo (кот → коты)
    - hundo → hundozo (собака → собаки)
    - infano → infanozo (ребёнок → дети)
    - amiko → amikozo (друг → друзья)
    - patro → patrozo (отец → отцы)
    - matro → matrozo (мать → матери)

## Особенности образования множественного числа

Окончание **-zo** добавляется ко всем существительным для образования множественного числа

Прилагательные не изменяются по числам в Globino, они остаются в одной и той же форме:
    - bela domo → bela domozo (красивый дом → красивые дома)
    - nova libro → nova librozo (новая книга → новые книги)
    - granda hundo → granda hundozo (большая собака → большие собаки)

## Множественное число с дополнениями

При использовании дополнений (с окончанием -n) в множественном числе, окончания сочетаются:
    - domon → domozon (дом → дома, в винительном падеже)
    - libron → librozon (книгу → книги, в винительном падеже)
    - katiton → katozon (котёнка → котят, в винительном падеже)

## Примеры предложений с множественным числом

1. "Mi vidas domozo" - Я вижу дома
2. "Ŝi legas librozo" - Она читает книги
3. "La katozo ludas" - Коты играют
4. "Niaj amikozo venis" - Наши друзья пришли
5. "La infanozo estas felicxa" - Дети счастливы

## Исключения

В Globino нет исключений в образовании множественного числа. Всегда используется окончание **-zo**, **-zon**.

## Дополнительная информация

В Globino нет грамматического рода, поэтому формы множественного числа одинаковы для всех родов
Прилагательные не изменяются по числам, что упрощает грамматику
Окончание -zo может использоваться и с прилагательными как существительными: "la belazo" (красивые люди/красивые вещи)
