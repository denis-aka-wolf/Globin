# Урок 15: Числительные в Globino

## Цели урока
- Понять систему числительных в Globino
- Научиться использовать количественные, порядковые, дробные и другие виды числительных
- Изучить образование составных числительных
- Практически отработать употребление числительных

## Введение
Числительные в языке Globino образуют мощную и логичную систему, которая позволяет выражать любые числовые значения с помощью небольшого количества базовых корней и суффиксов. В отличие от многих национальных языков, в Globino существует стройная система образования числительных, основанная на математическом принципе.

## Базовые корни числительных

### Основные числа от 0 до 10:
- **0** - **zo** (ноль)
- **1** - **li** (один)
- **2** - **na** (два)
- **3** - **vi** (три)
- **4** - **so** (четыре)
- **5** - **tu** (пять)
- **6** - **ra** (шесть)
- **7** - **ke** (семь)
- **8** - **mo** (восемь)
- **9** - **fa** (девять)
- **10** - **ti** (десять)

### Основные разряды:
- **100** - **sen** (сто)
- **1000** - **mil** (тысяча)
- **10^6** - **lon** (миллион)
- **10^9** - **ard** (миллиард)
- **10^12** - **rio** (триллион)
- **10^15** - **zor** (квадриллион)

## Количественные числительные

Количественные числительные отвечают на вопрос "сколько?" и обозначают количество предметов.

### Простые числа:
- **li** (один)
- **na** (два)
- **vi** (три)
- **so** (четыре)
- **tu** (пять)

### Составные числа:
Для образования составных чисел используются базовые корни:
- **11** - **tili** (10+1) 
- **12** - **tina** (10+2)
- **13** - **tivi** (10+3)
- **14** - **tiso** (10+4)
- **15** - **titu** (10+5)
- **20** - **nati** (2×10)
- **30** - **viti** (3×10)
- **40** - **soti** (4×10)
- **50** - **tuti** (5×10)
- **100** - **sen** (сто)
- **200** - **nasen** (2×100)
- **1000** - **mil** (тысяча)
- **2000** - **namil** (2×1000)

### Примеры составных чисел:
- **21** - **natili** (21)
- **22** - **natina** (22)
- **35** - **vititu** (35)
- **123** - **sen-nativi** (100+23)
- **1567** - **mil-tusen-rati** (1000+500+67)

## Порядковые числительные

Порядковые числительные образуются с помощью суффикса **-o**:

### Примеры:
- **li** (один) → **lio** (первый)
- **na** (два) → **nao** (второй)
- **vi** (три) → **vio** (третий)
- **so** (четыре) → **soo** (четвертый)
- **ti** (десять) → **tio** (десятый)
- **sen** (сто) → **seno** (сотый)
- **mil** (тысяча) → **milo** (тысячный)

## Дробные числительные

Дробные числительные образуются с помощью суффикса **-pa** (часть):

### Примеры:
- **na** (два) → **napa** (половина, 1/2)
- **vi** (три) → **vipa** (треть, 1/3)
- **so** (четыре) → **sopa** (четверть, 1/4)
- **tu** (пять) → **tupa** (пятая часть, 1/5)
- **ti** (десять) → **tipa** (десятая часть, 1/10)
- **sen** (сто) → **senpa** (процент, 1/100)
- **mil** (тысяча) → **milpa** (промилле, 1/1000)

### Сложные дроби:
- **3/4** - **vi sopa** (три четверти)
- **2/5** - **na tupa** (две пятых)
- **7/8** - **ke mopa** (семь восьмых)

## Кратные числительные

Кратные числительные образуются с помощью суффикса **-mo** (умножить):

### Примеры:
- **na** (два) → **namo** (вдвойне, двойной)
- **vi** (три) → **vimo** (втройне, тройной)
- **so** (четыре) → **somo** (вчетверо, четверной)
- **ti** (десять) → **timo** (десятикратный)
- **sen** (сто) → **senmo** (сто кратный)

## Собирательные числительные

Собирательные числительные образуются с помощью суффикса **-to** (вместе, группа):

### Примеры:
- **na** (два) → **nato** (пара, вдвоем)
- **vi** (три) → **vito** (тройка, втроем)
- **so** (четыре) → **soto** (четверка, вчетвером)
- **tu** (пять) → **tuto** (пятерка, впятером)

## Наречия числительные

Для вводных слов при перечислении используются наречия с суффиксом **-oe**:

### Примеры:
- **lio** (первый) → **lioe** (во-первых)
- **nao** (второй) → **naoe** (во-вторых)
- **vio** (третий) → **vioe** (в-третьих)
- **soo** (четвертый) → **sooe** (в-четвертых)
- **tio** (десятый) → **tioe** (в-десятых)

## Существительные числительные

Существительные числительные образуются с помощью суффикса **-o**:

### Примеры:
- **ti** (десять) → **tio** (десяток)
- **sen** (сто) → **seno** (сотня)
- **mil** (тысяча) → **milo** (тысяча как сущ.)

## Десятичные дроби

Для десятичных дробей используется слово **"to"** (точка):

### Примеры:
- **0.5** - **zo-to-tu** (ноль точка пять)
- **3.14** - **vi-to-li-so** (три точка один четыре)
- **0.001** - **zo-to-zo-zo-li** (ноль точка ноль ноль один)

## Примеры предложений с числительными

1. **Mi havas tu hundozn.** (У меня пять собак.)
2. **Si estas lio en klaso.** (Она первая в классе.)
3. **Gi estas napo de pizo.** (Это половина пиццы.)
4. **Ili iris nato.** (Они пошли вдвоем.)
5. **Vito personoz staras kune.** (Трое людей стоят вместе.)
6. **Lioe, ni devas komenci.** (Во-первых, мы должны начать.)
7. **Li ricevis senon el sen.** (Он получил сто из ста.)
8. **Si estas nao plej bona.** (Она вторая лучшая.)

## Особенности использования числительных в Globino

1. Числительные не изменяются по родам и числам
2. Существительное после числительного не требует артикля
3. Порядковые числительные используют окончание как прилагательные (-a)
4. Суффиксы позволяют легко образовывать различные виды числительных

## Дополнительная информация
- Система числительных Globino основана на математическом конструкторе
- С помощью базовых корней и суффиксов можно выразить любое числовое значение
- Суффиксы: -o (порядковые), -pa (дробные), -mo (кратные), -to (собирательные), -oe (вводные наречия)
- Числительная система Globino значительно упрощает работу с числами по сравнению с другими языками
