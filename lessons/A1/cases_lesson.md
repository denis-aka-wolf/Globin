## Введение

В языке Globino система падежей очень проста и логична. В отличие от многих национальных языков, в Globino используется минимальное количество падежей. Основной падеж, который имеет специальное окончание, - это винительный падеж (дополнение), который обозначается окончанием **-n**.

## Основные особенности падежной системы Globino

### 1. Именительный падеж (подлежащее)

Именительный падеж - это основная форма слова, которая не имеет специального окончания. Используется для обозначения подлежащего в предложении.

Примеры:
    - **Mi** estas studento. (Я студент.)
    - **Hundo** dormas. (Собака спит.)
    - **Libro** estas interesa. (Книга интересная.)

### 2. Винительный падеж (дополнение)

Винительный падеж обозначается окончанием **-n** и используется для обозначения прямого дополнения в предложении. Это слово, которое напрямую испытывает на себе действие глагола.

Примеры:
    - Mi amas **vin**. (Я люблю вас.)
    - Hundo vidis **katon**. (Собака видела кошку.)
    - Si legas **libron**. (Она читает книгу.)

## Правила образования винительного падежа

Для образования винительного падежа к существительному, прилагательному или местоимению добавляется окончание **-n**:

### Существительные

- domo → domon (дом → дом (винительный))
- libro → libron (книга → книгу)
- hundo → hundon (собака → собаку)
- kato → katon (кошка → кошку)

### Прилагательные

- bela → belan (красивый → красивую (винительный))
- granda → grandan (большой → большую (винительный))
- bona → bonan (хороший → хорошую (винительный))

### Местоимения

- min (меня)
- vin (тебя, вас)
- lin (его)
- sin (ее)
- gin (его/её для неодушевленных)
- nin (нас)
- ilin (их)

## Примеры предложений с падежами

### Подлежащее (именительный падеж) + дополнение (винительный падеж)

1. **Mi** (именительный) amas **vin** (винительный). (Я люблю вас.)
2. **Infano** (именительный) vidis **hundon** (винительный). (Ребенок видел собаку.)
3. **Si** (именительный) acetas **belegan libron** (винительный). (Она покупает красивую книгу.)
4. **Patro** (именительный) instruas **infanon** (винительный). (Отец обучает ребенка.)
5. **Ili** (именительный) rigardas **belan domon** (винительный). (Они смотрят на красивый дом.)

## Особенности использования падежей в Globino

### 1. Свобода порядка слов

Благодаря использованию окончания -n для дополнения, в Globino возможна свобода в расположении слов в предложении:

- "Mi vidas hundon" = "Hundon mi vidas" = "Hundon mi vidas" (Я вижу собаку)

### 2. Падежи и прилагательные

Когда прилагательное относится к дополнению, оно также принимает окончание -n:

- "Mi vidis belan hundon" (Я видел красивую собаку)
- "Si amas malgrandan domon" (Она любит маленький дом)

### 3. Падежи и множественное число

Окончание -n может сочетаться с окончанием множественного числа -zo:

- "Mi renkontis studentozon" (Я встретил студентов)
- "Si instruas katozon" (Она обучает кошек)

## Дополнительная информация

В Globino нет сложной системы падежей как в русском или латинском языке
Окончание -n используется только для обозначения винительного падежа (прямого дополнения)
Благодаря этой системе возможна свобода в порядке слов в предложении
Косвенные дополнения (с предлогами) не получают окончания -n
