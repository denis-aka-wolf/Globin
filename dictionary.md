# Словарь Globin

## A

### Am-

Amo - 1)сущ. любовь 2)сущ. первая буква алфавита Globin
Ami - любить
Amo — любовь (существительное)
Ami — любить (глагол, инфинитив)
Ama — любовный, любимый (прилагательное)
Ame — с любовью, любовно (наречие) 
2. Глагольные формы (времена и наклонения)
Amas — люблю / любит (настоящее время)
Amis — любил / любила (прошедшее время)
Amos — буду любить / полюблю (будущее время)
Amus — любил бы (условное наклонение)
Amu — люби! / пусть любит (повелительное наклонение / волитив) 
3. Множественное число и падеж
Amozo — чувства любви (множественное число)
Amon — любовь (винительный падеж, объект действия, например: Mi sentas amon — «Я чувствую любовь»)
Amojn — любови (множественное число + винительный падеж) 
4. Причастия (активные и пассивные)
Amanta — любящий (настоящее время)
Aminta — любивший (прошедшее время)
Amonta — тот, кто полюбит (будущее время)
Amata — любимый (кем-то в данный момент)
Amita — возлюбленный (тот, кого уже полюбили) 
5. Популярные производные с суффиксами
Amiko — друг (дословно «человек, связанный любовью/дружбой»)
Amindumi — ухаживать, любезничать
Amema — любящий, склонный к любви
Malami — ненавидеть (приставка mal- меняет значение на противоположное) 

Amozeal- алфавит
