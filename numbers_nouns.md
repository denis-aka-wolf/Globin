# Существительные числительные в Глобино

## Общее описание

В отличие от служебных числительных (Ти, Сен, Мил), существительные в Глобино имеют окончание **-а** (стандарт для имен существительных во многих логических языках). Это позволяет склонять их или использовать как самостоятельные объекты.

## Формирование существительных числительных

Для образования существительных числительных используется суффикс **-а**:
- Основа числительного + **-а** = существительное

### Примеры на основе чисел:

| Число | Существительное | Перевод | Пример | Перевод примера |
|-------|----------------|---------|--------|-----------------|
| Ти (10) | Тиа | Десяток | Ladi na tia ovo | Купи два десятка яиц |
| Сен (100) | Сена | Сотня | Tuta sena problemo | Целая сотня проблем |
| Мил (1000) | Мила | Тысяча | | |
| Лон (1000000) | Лона | Миллион | | |

## Разница между формами

- **На-ти** — число 20
- **На тиа** — два десятка (две упаковки по 10)
