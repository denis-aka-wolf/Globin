# Наречия числительные в Глобино

## Общее описание

Для вводных слов, используемых при перечислении, мы используем порядковое числительное (суффикс **-о**) и добавляем наречный суффикс **-е** (традиционно для наречий).

**Формула:** Число + **-о** + **-е** = **-ое**

## Формирование наречий

| Порядковое числительное | Наречие | Перевод |
|------------------------|---------|
| Лио (первый) | Лиое | Во-первых |
| Нао (второй) | Наое | Во-вторых |
| Вио (третий) | Виое | В-третьих |
| Соо (четвертый) | Соое | В-четвертых |
| Тио (десятый) | Тиое | В-десятых |

## Примеры использования

**Lioе, se ladi niso. Naoе, se ladi voso.**
(Во-первых, это дорого. Во-вторых, это опасно.)
